Morse Code Translater

If you wish to send a message and talking is not the best option here is how you can use a morse code translator:
  1. Type the message you wish to deliver into the box
  2. Press translate
  3. Look for the button that corresponds to your situation. Click your mouse based on the morse code that appears, so the screen flashes in sync with the code durations.
  4. Note: dot = 1 sec, bar = 3 secs
